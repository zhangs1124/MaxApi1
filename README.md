# MAX USDT TWD 定時報價 (Chrome 擴充功能)

定時取得 MAX 交易所 (加密貨幣) 與台灣股市 (如 0050, 0056, 2330) 的報價，並提供豐富的提醒與自訂追蹤功能。

## ✨ 主要功能

- **定時/手動更新報價**：支援設定固定週期（分鐘）自動抓取最新價格，也能隨時點擊「立即更新」重新取得報價。
- **多元交易對支援**：可自行輸入包含加密貨幣（如 `btcusdt`, `usdttwd`）與台股代號（如 `2330`, `0050`），所有代號將自動對應顯示對應中文名稱。
- **價格警報（示警功能）**：
  - 支援針對每一個交易對設定專屬的「上限（高標）」與「下限（低標）」觸發數字。
  - **雙重通知機制**：
    1. **原生系統推播**：條件觸發時，由 Chrome 喚起 Windows/Mac 原生系統桌面橫幅通知。
    2. **網頁 Alert 警告**：為防止系統通知被防擾機制捕捉，更進一步自動追蹤您當前正在瀏覽的分頁，並硬性注射 `alert()` 警告視窗，確保緊急訊號不漏接！
  - 成功觸發價格警告後，系統為防止連續轟炸，會自動將該交易對的追蹤「取消啟用」，直至您再次打勾。
- **擴充功能圖示通知**：報價更新時，會自動在擴充功能的 ICON 上方跳出 Badge （角標提示）標示最新動態。

## 🚀 安裝與使用方式

1. 在 Chrome 瀏覽器輸入 `chrome://extensions/` 開啟擴充功能管理。
2. 開啟右上角的「開發人員模式 (Developer mode)」。
3. 點選「載入未封裝項目 (Load unpacked)」，選擇本專案所在資料夾。
4. 釘選該外掛的按鈕於工具列上，點一下即可檢視與設定所有監控對象。

## 簡易使用書（新增交易對 + 上下限示警）

### 1. 先打開外掛介面
點一下擴充功能圖示（工具列上的按鈕），會看到外掛的設定介面；示警觸發時也會嘗試在「目前分頁」跳出 `alert()`，因此你的 Chrome 需要保持開著。

### 2. （可選）開啟「通知」
介面裡有一個「通知」勾選框：
- 勾選後，當「這次更新沒有觸發示警」時，才會用 Windows/Mac 原生系統通知你「報價更新完成」。
- 真正觸發「價格示警」時，外掛也會嘗試跳出系統通知與 `alert()`（但是否能顯示仍取決於你是否允許 Chrome 的通知權限）。

### 3. 新增交易對（虛擬幣或股票代碼）
在「新增交易對」輸入框輸入代號，支援格式如下，然後按「加入」：
- 虛擬幣交易對：`btcusdt`、`usdttwd`（也可以輸入 `BTC USDT` 之類含空格/符號的寫法，外掛會自動整理成標準格式）
- 台股代號：`2330`、`0050`、`0056` 等（也可輸入 `2330.TW`）

補充：輸入框是可接受逗號/空白分隔的多個字串（例如 `BTCUSDT, 2330`），外掛會整理成唯一交易對後加入清單。

### 4. 設定上限（高標）/下限（低標）與「啟用」
在交易對清單中，每一列都有三個控制項：
- `上限（上方輸入框）`：當「最新價格」`last` >= 上限時觸發
- `下限（下方輸入框）`：當「最新價格」`last` <= 下限時觸發
- `啟用` 勾選：是否啟用這個交易對的示警

你可以只填上限、不填下限；或只填下限、不填上限（留空代表不觸發那一邊的條件）。
另外，上下限的輸入與勾選一改就會立即保存，不需要再按「保存」。

### 5. 手動測試（或等排程）
- 點 `立即更新` 可以立刻抓取最新價格並檢查是否觸發示警
- 或者照你介面上的排程設定（固定間隔 / 每日時間）等系統自動更新

### 6. 觸發示警後會發生什麼事
當任何交易對觸發「價格示警」時，外掛會同時做以下事情（雙重提醒）：
1. **系統通知**：在桌面跳出原生通知（標題含「價格觸發示警」）
2. **網頁訊息**：嘗試在你「目前正在看的分頁」注入 `alert()`（如果你正在看的是 `chrome://` 之類頁面，可能不會注入）

同時，為了避免連續轟炸：觸發成功後，該交易對的 `啟用` 會被外掛自動取消（你需要再重新勾回去，才會再次觸發）。

## 🛠 開發細節 (Manifest V3)

本專案完全遵循最新的 Manifest V3 架構開發：
- 以 `chrome.alarms` 取代 `setInterval`，避免 Service Worker 休眠影響排程。
- 完善整合跨內容腳本溝通機制 (`chrome.runtime.sendMessage` & `chrome.scripting.executeScript`)。
- 使用 `chrome.storage.sync` 作為全域設定保存，確保狀態穩定同步。
