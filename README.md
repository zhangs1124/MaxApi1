# MAX USDT TWD 定時報價 (Chrome 擴充功能)

定時取得 MAX 交易所 (加密貨幣) 與台灣股市 (如 0050, 0056, 2330) 的報價，並提供豐富的提醒與自訂追蹤功能。

## ✨ 主要功能

- **定時/手動更新報價**：支援設定固定週期（分鐘）自動抓取最新價格，也能隨時點擊「立即更新」重新取得報價。
- **多元交易對支援**：可自行輸入包含加密貨幣（如 `btcusdt`, `usdttwd`）與台股代號（如 `2330`, `0050`），所有代號將自動對應顯示對應中文名稱。
- **價格警報（示警功能）**：
  - 支援針對每一個交易對設定專屬的「上限（高標）」與「下限（低標）」觸發數字。
  - **雙重通知機制**：
    1. **原生系統推播**：條件觸發時，由 Chrome 喚起 Windows/Mac 原生系統桌面橫幅通知。
    2. **網頁 Alert 警告**：為防止系統通知被防擾機制捕捉，更進一步自動追蹤您當前正在瀏覽的分頁，並硬性注射 `alert()` 警告視窗，確保緊急訊號不漏接！
  - 成功觸發價格警告後，系統為防止連續轟炸，會自動將該交易對的追蹤「取消啟用」，直至您再次打勾。
- **擴充功能圖示通知**：報價更新時，會自動在擴充功能的 ICON 上方跳出 Badge （角標提示）標示最新動態。

## 🚀 安裝與使用方式

1. 在 Chrome 瀏覽器輸入 `chrome://extensions/` 開啟擴充功能管理。
2. 開啟右上角的「開發人員模式 (Developer mode)」。
3. 點選「載入未封裝項目 (Load unpacked)」，選擇本專案所在資料夾。
4. 釘選該外掛的按鈕於工具列上，點一下即可檢視與設定所有監控對象。

## 🛠 開發細節 (Manifest V3)

本專案完全遵循最新的 Manifest V3 架構開發：
- 以 `chrome.alarms` 取代 `setInterval`，避免 Service Worker 休眠影響排程。
- 完善整合跨內容腳本溝通機制 (`chrome.runtime.sendMessage` & `chrome.scripting.executeScript`)。
- 使用 `chrome.storage.sync` 作為全域設定保存，確保狀態穩定同步。
